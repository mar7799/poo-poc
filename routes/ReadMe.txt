This is sample project to demostrate the mongodb interaction using node.js. 
This is POO Schema where we have users, products, orders and payment.

we have multiple endpoints to interact with users collection in MongoDb likewise for the others.
List of Endpoints:
--/users/registerUser - to insert a new user.
--/users/registerUsers - to insert list of users.
--/users/getUsers - to view list of registered users.
--/orders/placeOrder - to place an order.
--/getCart - to see list of orders in cart.
--/products/addProduct - to add product into db.
--/products/getProducts - to see list of products.
